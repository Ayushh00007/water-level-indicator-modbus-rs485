# 🌊 Water Level Indicator System using Modbus RTU (RS485)

## 📌 Project Overview
This project is an industrial-style water level monitoring system using **Modbus RTU communication over RS485**.  
It enables long-distance, noise-resistant data transmission between slave sensor node and master controller.

---

## ⚙️ System Architecture
- **Modbus Slave:** Reads water level sensor data
- **RS485 Communication:** Transfers data in industrial protocol
- **Modbus Master (Python):** Receives and displays real-time data

---

## 🧠 Key Features
- Real-time water level monitoring
- Industrial communication protocol (Modbus RTU)
- RS485 long-distance communication
- Python-based monitoring dashboard
- Low-cost automation solution

---

## 💻 Files Included
